	5. SIP 폰 셋팅  
		(1) 웹 관리자 페이지 활성화  
			(1)-1. ‘전환’ 버튼 누름
			(1)-2. ‘1번’ 눌러서 ‘1. 네트워크’ 메뉴 이동
			(1)-3. ’1번’, ‘2번’ 연속으로 눌러서 ’12. WEB : HTTPS’ 메뉴 이동
			(1)-4. 스피커 볼륨 버튼으로 HTTPS가 화면에 표시되도록 놓고 ‘보류’ 버튼 클릭

		(2) 관리자 페이지 패스워드 변경  
			(2)-1. ‘전환’ 버튼 누름
			(2)-2. ‘1번’ 눌러서 ‘1. 네트워크’ 메뉴 이동
			(2)-3. ‘2번’ 눌러서 ‘2. 아이피 주소’ 메뉴 이동하여 현재 sip 폰에 할당된 주소를 확인.
			(2)-4. sip폰과 같은 네트워크에 있는 피시 또는 휴대폰에서 이전 단계에서 확인한 아이피 주소를 브라우저에 입력한다.
				*만약 브라우저에서 ‘연결이 비공개로 설정되어 있지 않습니다.’라는 페이지가 표시된다면, ‘세부정보 표시’ 클릭후 ‘안전하지 않음으로 이동’을 클릭한다.
			(2)-5. ‘Login’ 버튼을 클릭하고 ID: user, PW: lip 를 입력한다.
			(2)-6. 왼쪽 사이드 메뉴에서 ‘전화기 설정’ -> ‘시스템’ 메뉴를 클릭
			(2)-7. ‘비밀번호 변경’ 섹션에서 ‘이전 비밀번호’에 lip 입력하고 ‘새로운 비밀번호’와 ‘비밀번호 재입력’ 필드에는 변경할 비밀번호(h7wideAA!@)를 입력하고 ‘변경’을 누른다.

		(3) 아이피 설정  
			* 왼쪽 사이드 메뉴의 ‘네트워크 설정’ -> ‘네트워크’ 메뉴를 클릭하면 고정 또는 유동 아이피, 아이피 및 서브넷 마스크, 게이트웨이를 설정할 수 있다.

		(4) VoIP 설정  
			(4)-1. 왼쪽 사이드 메뉴의 ‘VoIP 설정’ -> ‘VoIP’ 메뉴를 클릭한다.
			(4)-2. ‘서버’ 섹션의 ‘프록시 SIP 서버 주소’, ‘보조 SIP 서버 주소’를 단지내 할당된 무인택배 서버 주소로 입력한다.
			(4)-3. 같은 페이지 내, ‘기능’ 섹션의 ‘링 무응답 시간(초)’, ‘Alert 무응답 시간(초)’ 옵션에 30을 입력하고 페이지 최하단의 ‘저장’ 버튼을 클릭한다.

		(5) VoIP 회선 설정 
			(5)-1. 왼쪽 사이드 메뉴의 ‘VoIP 설정’ -> ‘VoIP 회선’ 메뉴를 클릭한다.
			(5)-2. SIP 서버 섹션의 ‘주소’ 필드에 할당된 무인택배 서버 주소로 입력한다.
			(5)-3. ‘화면표시 이름’ 필드에는 ‘관리실’을 입력
			(5)-4. ‘계정’ 필드와 인증이름에 ‘1000’을 입력한다. (Asterisk 서버에 1000 계정이 등록되어 있어야 한다.)
			(5)-5. ‘암호’ 필드에 ‘1234’ 입력(Asterisk 서버에 1000 계정 암호가 ‘1234’가 등록되어 있어야 한다.)
			(5)-6. ‘Extension 번호’ 필드에 ‘1000’ 입력하고 페이지 최하단의 ‘저장’ 버튼 클릭

		(6) 설정 완료후 전화기 재시작  
			* 관리자 페이지의 사이드 메뉴에서 ‘상태’ -> ‘다시 시작’ 메뉴를 클러서 재시작하거나 전원을 리셋해서 전화기를 재시작한다.
