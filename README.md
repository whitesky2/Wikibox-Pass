# 개발 기획 관련
	- 세탁 프로세스
		1. 사용자가 앱에서 세탁을 선택한다.
		2. 사용자가 세탁물을 '보관함에 보관'할지 세탁업체가 방문해서 '수거'해 줄지 선택한다.
		
		3-1-1. 사용자가 앱에서 '보관함에 보관'을 선택한 경우
		3-1-2. 앱에서 주변 보관함의 id를 스캔한다.
		3-1-3. 보관함 id가 스캔되면 보관함 맵이 보이고 세탁물을 보관할 함을 선택
		3-1-4. 보관함 문이 열리면 세탁물을 넣고 문을 닫는다.

		3-2-1. 사용자가 앱에서 '수거'를 선택한다.
		3-2-2. 사용자가 집주소, 방문일, 방문시간, 요청사항(텍스트, 사진) 앱에 입력하고 확인을 선택한다.
		
		4. 수거 업체에서 세탁 데이터를 확인하고, 해당 건이 '보관함에 보관'인지, '수거' 요청 건인지 체크
		
		5-1-1. 세탁 신청건이 '보관함에 보관'인 경우, 해당 보관함 앞으로 이동해서 세탁 수거 앱을 실행.
		5-1-2. 세탁 수거 앱에서 해당 세탁 데이터를 선택한 후, '수거' 버튼을 클릭.
		5-1-3. 보관함 문이 열리면 세탁물을 수거한다.
		
		5-2-1. 세탁 신청 건이 '수거'일 경우, 사용자가 입력한 주소로 이동하여 세탁물을 수거한다.
		5-2-2. 수거 이후 관리자 페이지에서 세탁 데이터의 상태를 '수거완료'로 변경
		
		6. 수거업체에서 세탁물을 확인한 뒤 손상된 부분을 사진을 찍어 세탁 수거앱으로 사진을 등록.
		7. 수거업체에서 사진 등록 뒤, 사진에 대한 코멘트를 관리자 앱으로 입력하고 '발행' 버튼을 입력해 인수증을 발행
		8. 사용자가 인수증을 확인하고 결제한다.
		(앱에서 결제하거나 인수증 문자 내에 있는 결제 링크를 클릭해서 결제한다.)
		
		9-1-1. 세탁이 완료된 이후 해당 세탁 건이 보관함에 보관된 건이면, 처음 보관되었던 보관함으로 이동해 세탁 수거 앱을 실행한다.
		(고객이 특별히 집으로 배송해달란 요청이 없을 경우)
		9-1-2. 세탁 수거 앱을 실행 후, 해당 세탁 건을 클릭하고 '보관' 버튼을 클릭한다.
		9-1-3. 보관함 문이 열리면 세탁이 완료된 세탁물을 넣고 문을 닫은 뒤 세탁 프로세스를 종료.
